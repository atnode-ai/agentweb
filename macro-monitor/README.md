# Macro Monitor

Automated weekly macroeconomic executive briefings — inflation, labour, growth, housing, sentiment, central-bank policy and financial conditions.

**Live:** https://atnode-ai.github.io/agentweb/macro-monitor/

- `index.html` — landing page + weekly report archive + detailed blackboard archive
- `reports/YYYY-Www.html` — weekly business briefings (ISO week)
- `blackboard/YYYY-Www.html` — full swarm data snapshots behind each briefing
- `architecture.html` — the agent-swarm architecture
- `tools/` — page builders used by the weekly automation

Published automatically every Saturday by the Macro Monitor swarm. _Internal monitoring summary — not investment advice._
